### Public release: v0.2.0

- **Feature:** Ability to use a single remote or local file (a recipe file).
    - Users can reference a single file from both local and remote sources (no support for chaining sources yet.)

- **Feature:** Selective uninstallation.
    - Users have the option to uninstall specific packages in one shot.

- **Bug Fixes and Improvements:**
    - Addressed various minor bugs and improved overall performance.

## Midterm release (feature complete release)

### Release: v1.0.0

- **Feature:** Support for chaining multiple remote and local recipe files.
    - Users have the flexibility to chain together multiple files, whether they are remote or local, for more complex operations.

- **Feature:** Uninstall by chaining multiple remote and local recipe files.
    - Same recipe files can be reused to nuke everything in one shot, reversing the installation process.

- **Compatibility and Stabilisation:**
    - Resolved compatibility issues with various file OSses and optimized file handling processes.

- **Binary installation wizard for macOS and Linux:**

## Long-term release

### Release: v1.5.0

- **Feature:** Enhanced security measures.
    - Implementation of advanced security protocols to safeguard file fetching and processes execution.

- **Feature:** Intelligent recommendations and profiles support.
    - Utilizing machine learning to suggest optimal file chains and uninstall sequences based on usage patterns.

- **Improvement:** User interface revamp.
    - Redesigned user interface for a more intuitive and seamless experience.

- **Bug Fixes and Improvements:**
    - Continued monitoring and refinement of features for maximum reliability.
